Download the CICIDS 2017 dataset from Kaggle and place the CSV here as network_logs.csv.

Link: https://www.kaggle.com/datasets/cicdataset/cicids2017

After downloading, rename the file to:
  network_logs.csv

Then run the batch analyzer:
  python src/data/batch_runner.py

(Make sure your FastAPI server is running first: python run.py)
